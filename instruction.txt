Необходимо иметь:
1. MySQL Server (делал на 5.7.17)
2. JDK 8
3. Apache Maven (3.3.9)

Что делать:
1. Настроить MySQL:
    1.1. Учетная запись:
        1.1.1. Если есть возможность, создать локальное подключение с логином и паролем "root"
        1.1.2. В противном случае, изменить username и password в файле src/main/resources/application.properties
    1.2. Схема и таблица:
        1.2.1. В папке src/main/resources/sql выполнить скрипт из файла script.sql
               (в командной строке выполнить "mysql < script.sql")
        1.2.2. Можно вручную заполнить, либо скомпилировать файл CreateData.java из пакета createData.
               Там можно указать количество записей, которые будут добавлены (по-умолчанию, один миллион)

        ПРИМЕЧАНИЕ: В файле CreateData, возможно, необходимо поменять USER и PASS
